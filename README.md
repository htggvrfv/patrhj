# Patr-keepalive

利用Github Actions，定时重启Patr云的部署以让应用稳定运行

## 部署说明

* Fork 这个项目，然后启动
* 项目用到的环境变量
  | 变量名 | 是否必须 | 默认值 | 备注 |
  | ------------ | ------ | ------ | ------ |
  | PATR_USERNAME | 是 | 无 | Patr 账户登录邮箱 |
  | PATR_PASSWORD | 是 | 无 | Patr 账户登录密码 |
